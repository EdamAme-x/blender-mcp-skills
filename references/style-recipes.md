## Industrial hero still

Use this when the subject is a product, mech part, hard-surface prop, or machine.

- Camera:
  - `50mm` to `85mm`
  - Slightly low angle
  - Moderate DOF, not extreme blur
- Lighting:
  - Hard key from one side
  - Soft fill kept low
  - Thin rim to separate silhouette
- Materials:
  - Roughness variation is mandatory
  - Use edge wear only on contact logic
  - Avoid universal grunge overlays
- Composition:
  - Large hero shape centered or rule-of-thirds anchored
  - Supporting props stay clearly subordinate
- Search terms:
  - `industrial product hero lighting`
  - `hard surface studio photography`
  - `commercial product rim light`
- Fail signs:
  - Looks like a CAD viewport
  - Every edge has the same wear
  - Background contrast equals hero contrast

## Moody cinematic environment

Use this when environment atmosphere carries the shot.

- Camera:
  - `24mm` to `40mm`
  - Low or eye-level angle depending on scale
  - Deep enough focus to read space layering
- Lighting:
  - One clear narrative source
  - One controlled accent source
  - Negative fill or occlusion to keep depth
- Materials:
  - Keep material families limited
  - Favor macro breakup over micro noise
- Composition:
  - Foreground frame
  - Midground hero zone
  - Background story support
- Search terms:
  - `cinematic environment concept frame`
  - `volumetric alley lighting`
  - `atmospheric environment still`
- Fail signs:
  - Fog is hiding weak layout
  - Bright spots are scattered without hierarchy
  - No readable depth after desaturation

## Stylized character portrait

Use this when the character face, pose, or costume is the main read.

- Camera:
  - `65mm` to `105mm`
  - Eye line or slightly above
  - Shallow DOF only if eyes remain perfectly readable
- Lighting:
  - Soft key with strong cheek and brow shaping
  - Hair or shoulder rim
  - Background kept simple
- Materials:
  - Skin, fabric, metal accessories must separate clearly
  - Stylization should simplify, not flatten
- Composition:
  - Face or upper body dominates
  - Pose line and silhouette must read instantly
- Search terms:
  - `stylized portrait lighting`
  - `character key art portrait`
  - `3d character bust render`
- Fail signs:
  - Hair merges into background
  - Eyes are not the sharpest read
  - Costume detail overwhelms face hierarchy

## Looping motion teaser

Use this when a short looping shot must feel polished fast.

- Camera:
  - Fixed or very small controlled move
  - Avoid complex handheld imitation unless the loop seam is designed
- Motion:
  - One primary motion idea only
  - Seam must be invisible in value, position, and timing
- Lighting:
  - Stable main light
  - Animated lights only if they support the loop concept
- Materials:
  - Prefer readable broad behavior over subtle microdetail
- Composition:
  - Loop should read from any entry frame
  - Motion should reinforce the hero instead of distracting from it
- Search terms:
  - `3d looping teaser animation`
  - `seamless cinematic loop`
  - `product loop animation`
- Fail signs:
  - Loop seam is visible only because framing changes
  - Camera motion does more work than subject motion
  - Motion blur is covering timing problems

## Recipe selection

- If the subject is object-first, start with `Industrial hero still`.
- If the mood of the space is the product, start with `Moody cinematic environment`.
- If the face or costume sells the frame, start with `Stylized character portrait`.
- If delivery is short-form motion, start with `Looping motion teaser`.
- Pick one recipe as the base. Borrowing from multiple recipes is allowed only after the first pass is stable.
